# Contributing

This is a personal homelab project. Contributions are not expected, but issues and suggestions are welcome.

## Workflow

- `main` → production. Never push directly.
- `develop` → active development. Branch from here.

## Branch naming
feat/description
fix/description
docs/description
chore/description

## Commit convention

Follows [Conventional Commits](https://www.conventionalcommits.org):
feat: add new service
fix: correct env var resolution
docs: update README
chore: clean up docker-compose

## Pull Requests

- Always open PRs from `develop` → `main`
- One thing per PR
- Include what changed and why
