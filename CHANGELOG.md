# Changelog

All notable changes to this project will be documented here.

## [v1.0.0] — 2026-04-05

### Added
- Ubuntu Server 24.04 on Raspberry Pi 5 (ARM64)
- Docker 29.3.1 + Compose v5.1.1
- Static IP 192.168.1.100
- Cloudflare Tunnel — zero open ports
- Nginx Proxy Manager + Let's Encrypt wildcard SSL (*.hugohhm.dev)
- Beszel server and container monitoring
- Prometheus + Node Exporter system metrics
- Grafana Node Exporter Full dashboard
- Loki + Promtail log aggregation
- Uptime Kuma service availability monitoring
- Glances system resource API
- GitHub Actions self-hosted runner (ARM64)
- CI/CD pipeline — validate + deploy + health check + notify
- Ntfy self-hosted push notifications
- Pi-hole DNS ad-blocker
- Vaultwarden self-hosted password manager
- Homepage dashboard with live widgets and Docker status
- Homepage credentials via HOMEPAGE_VAR_* environment variables
